# Dialogue labeling — README

Небольшое, удобное Streamlit-приложение для разметки диалогов в формате **JSONL**.  
Позволяет просматривать диалоги, помечать `client_status` и `success`, автоматически сохранять изменения в сессии и скачивать размеченный файл.

https://dialogues-labeling-smnk-1.streamlit.app/
